FILL INSPECTOR NAME

Enter the inspector name to begin the inspection process.

FILL PART DATA

Enter from the label:
• Part number
• Serial number

CAPTURE PART LABEL

Capture image of part's label.

FILL CRITERIA DEFECT FORM

Select defect criteria:
• Defect type: (e.g. scratch, dent)
• Surface quality: (A/B/C)
• Finish quality.

CONFIRM CRITERIA DATA

Verify against acceptance criteria:
• Check if selections match defect conditions.
• YES = Log as defect.
• NO = False alarm.

CAPTURE CONTEXT PICTURE

Step back and capture:
• Defect and surrounding area.
• Ensure clear visibility of defects' location.

CAPTURE DEFECT

Take close-up picture of defect.
• Center flaw in the focused half.

REPEAT / KEEP / DROP

Take a close look at the defect picture.
• Repeat: if blurry/out-of-focus.
• Keep: if sharp and clear.
• Drop: to discard inspection.

NEW / ADD / PRINT

Final actions:
• New: Start new report on another part.
• Add: Log another defect of the same part.
• Print: Save current report.
